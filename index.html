Apache Web Server Project

Kali Linux Implementation

Project Overview

This project demonstrates the installation, configuration, and deployment of an Apache web server on a Kali Linux system. A custom-designed webpage is hosted locally using the Apache HTTP server, showcasing the fundamentals of web hosting and server management.

Technologies Used

  • Kali Linux Operating System
  • Apache2 Web Server
  • HTML5
  • CSS3
  • Git and GitHub

Implementation Steps

  • Installed Apache server using package manager
  • Started and enabled Apache service
  • Verified server using localhost
  • Modified default web directory
  • Designed and deployed custom webpage

Result

The Apache server successfully hosted a structured and styled webpage accessible through the local host environment. The project demonstrates the working of a web server and the deployment process in a Linux-based system.
